# IoT-device model
Talkpool OY1110

## Transmission type
LoRaWAN

## Description
Til celcius og luftfugtighed i procent. Lokation - hvis device er udpeget i OS2Iot-kortinterface medtages i payload-dekoder, herunder at geo-attributterne x og y (WGS 84) medtages i output.
